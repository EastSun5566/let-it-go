# Let it go

> ❄️ Make your website snow instantly

## ✨ Install

```sh
npm i let-it-go

# or yarn add let-it-go
```

## 🚀 Usage

### Basic

```ts
import { LetItGo } from "let-it-go";

// Just snow!
const snow = LetItGo();

// Just stop
snow.letItStop();

// And snow again!
snow.letItGoAgain();

// Clear canvas
snow.clear();
```
